快速接入示例 (Python)

只需更改 base_url，即可使用 2026 年最新的 GPT API 或 Claude API：

    base_url="https://jeniya.chat/v1",
    api_key="sk-你的简易API令牌"
)

    model="claude-3-7-sonnet-20250219-thinking", # 自动中转接入

print(response.choices[0].message.content)
